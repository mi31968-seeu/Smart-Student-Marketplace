# Smart Student Marketplace & Service Platform

## Overview

Smart Student Marketplace & Service Platform is a web application developed by Mustafa Ismaili and Amra Demiri.

The idea of the project is to help university students find and offer services in one organized platform. Students can use it to register, log in, post services, search for offers, and view reviews.

## Team Members

## Features

- Student registration
- Student login
- Service posting
- Service search and filtering
- Reviews and ratings
- Marketplace-style frontend
- Backend API
- Unit tests

## Setup

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/Smart-Student-Marketplace.git
```

Open the project folder:

```bash
cd Smart-Student-Marketplace
```

Install backend requirements:

```bash
cd backend
pip install -r requirements.txt
```

Run the backend:

```bash
python app.py
```

Open the frontend in a browser:

```text
frontend/index.html
```

Demo account:

```text
Email: student@example.com
Password: 12345
```

## Running Tests

From the main project folder, run:

```bash
pytest
```

The tests are in the `tests` folder.
