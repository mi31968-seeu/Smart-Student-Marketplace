# Project Requirements

## Functional Requirements

1. A student can create an account.
2. A student can log in.
3. A student can post a service.
4. A student can search and filter services.
5. A student can view reviews and ratings.
6. The platform shows team and project information.

## Non-Functional Requirements

1. The interface should be simple to use.
2. The project should have a clear folder structure.
3. The repository should include setup instructions.
4. The project should include unit tests.
5. The project should be stored in a GitHub repository.
